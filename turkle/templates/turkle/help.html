Help

Accept next Task

The task is assigned to you. If you do not finish it, you will be reminded on the main page that you have remaining open tasks. If you do not complete it by the deadline, it is returned to the task list.

Auto-accept next Task

Rather than being returned to the main page each time, the next task is automatically assigned to you.

Return Task

Returns the currently assigned task so that it is no longer assigned to you. If you accept a task from the same batch, that returned task could be assigned to you again. If you do not want to work on that particular task, you should skip it.

Skip Task

Skip to the next available task in the Batch. The task is removed from your assignments. You may see it again if all other tasks have been completed and no one else completed it.
